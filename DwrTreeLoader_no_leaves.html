DwrTreeLoader Example 2: No leaves, just folders

This example shows a simple tree built using a DwrTreeLoader. The only difference to example 1 is a slightly modified CSS style that causes nodes without children to be displayed as empty folders instead of leaves: 

.x-tree-node-leaf .x-tree-node-icon {
      background-image: url(../../resources/images/default/tree/folder.gif);
}

Note: The nodes are currently not fetched from a real DwrService, a "fake service" is used instead.
But the DwrTreeLoader behaves in exactly the same way as it would when using a normal DWR Service. A live DWR example is coming soon ;)
